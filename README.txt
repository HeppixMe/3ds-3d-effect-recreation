# 3DS 3D‑Effect Recreation

Dieses kleine Python‑Projekt nutzt OpenCV und Pygame,  
um die 3D funktion eines Nintendo 3Ds nachzumachen.

## Voraussetzungen

- Python 3.x
- OpenCV (`pip install opencv‑python`)
- Pygame (`pip install pygame`)

## Installation

```bash
git clone https://github.com/<dein‑user>/3ds-3d-effect-recreation.git
cd 3ds-3d-effect-recreation
pip install opencv-python pygame
